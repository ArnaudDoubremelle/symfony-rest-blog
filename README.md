# symfony-rest-blog

## Articles

All articles  
GET /api/articles

One article  
GET /api/articles/{id}

Create article  
POST /api/articles/new
param title, text, author, category

Edit article  
PUT /api/articles/{id}
param title, text, author, category

Delete article  
DELETE /api/articles/{id}
